# CodeAlpha Age Calculator

This is a web-based Age Calculator project created as part of the CodeAlpha Web Development Internship.

## Project Overview

The Age Calculator allows users to enter their date of birth and calculates their exact age in years, months, and days.

## Features

- User can select date of birth
- Calculates age in years, months, and days
- Prevents future date selection
- Shows validation messages
- Responsive and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Covered

- DOM Manipulation
- JavaScript Date Object
- Input Validation
- Responsive Web Design
